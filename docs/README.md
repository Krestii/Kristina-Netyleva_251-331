## ВВЕДЕНИЕ

В рамках проектной практики стояла задача создать многостраничный сайт для проекта «Живой стилистический словарь русского языка. Лингвистический интернет-сервис». Итоговый продукт должен был быть не только рабочим, но и готовым к публичному представлению: с единой системой стилей, удобной навигацией и структурированным контентом.

Практика проходила в период с 03 февраля 2025 г. по 24 мая 2025 г. под руководством куратора от кафедры «Информатика и информационные технологии».

---

## 1. ОБЩАЯ ИНФОРМАЦИЯ О ПРОЕКТЕ

**«Название проекта:** Project Spacewolf.

**Цель проекта:** Создать демо-версию игры жанра “космический шутер” с возможностью переключения между пилотами на лету, а также сайт-визитку.

**Задачи проекта:**
* Провести анализ актуальности проекта.
* Разработать и утвердить единую художественную концепцию.
* Построить тестовый уровень.
* Создать механики для полёта по заданному маршруту с возможностью переключения между пилотами.
* Наполнить демо-версию визуальными и звуковыми и т.д. элементами для достижения максимальной презентабельности.
* Освоить инструменты Git и Markdown.
* Создать сайт-визитку, рассказывающий об игре и участниках.

## 3. ОПИСАНИЕ ЗАДАНИЯ ПО ПРОЕКТНОЙ ПРАКТИКЕ

Обязательная часть: 
* Изучить синтаксис Markdown и вести всю документацию в этом формате.

## 4. ОПИСАНИЕ ДОСТИГНУТЫХ РЕЗУЛЬТАТОВ ПО ПРОЕКТНОЙ ПРАКТИКЕ

В ходе проектной практики были достигнуты следующие результаты:

### 4.1. Веб-сайт (HTML/CSS)

* Создан статический сайт из 5 страниц (index.html, about.html, participans.html, log.html, resources.html) с единым файлом стилей style.css.
* Дизайн выполнен в космической тематике: преимущественно тёмно-синий цвет для фона и таблиц, светло-голубой текст разных оттенков, элементы звёзд.
* Реализована адаптивная вёрстка (флексбоксы), сайт корректно отображается на компьютерах и мобильных устройствах.
* В журнал внесены пять записей о ключевых этапах проекта.

### 4.2. Документация в Markdown и Git

* Создан репозиторий на GitHub с файлами: README.md, task/README.md, report.md.
* Изучен синтаксис Markdown (заголовки, списки, таблицы).
* Настроена структура папок, выполнены коммиты с осмысленными сообщениями.

### 4.3. Личный вклад и приобретённые навыки

* HTML и CSS – освоила создание многостраничного сайта, использование flexbox для размещения элементов, подключение шрифтов, добавление изображений, а также оформление карточек участников и страницы журнала прогресса.
* Markdown и Git – научилась оформлять документацию в репозитории с помощью Markdown, а также работать с коммитами для управления изменениями проекта.
* Командная работа – распределяла задачи с другими участниками, занималась созданием дизайнов для главных персонажей видеоигры, а также участвовала в разработке интерфейса главного меню.
