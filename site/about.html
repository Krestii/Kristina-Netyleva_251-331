Информация о проекте

Актуальность

Отечественная игровая индустрия активно развивается. Несмотря на технический прогресс, игровое сообщество всё чаще сталкивается с проблемой однообразия игрового опыта.

В сегменте космических экшн-игр наблюдается определенная стагнация: большинство проектов либо следуют жесткой «рельсовой» структуре, либо предлагают чрезмерную свободу в пустом пространстве, лишенную внятной динамики.

Актуальность данного проекта заключается в создании продукта, который преодолевает эти жанровые шаблоны через инновационное проектирование уровней и уникальную визуальную подачу.

Проблематика

На данный момент космические шутеры являются крайне непопулярным жанром. Проблема современных научно-фантастических шутеров — их структурная предсказуемость. Игрокам не хватает проектов с глубокой проработкой игровых ситуаций, где каждый этап прохождения предлагает новый темп и задачи.

Цель

Создать демо-версию игры жанра космический шутер с возможностью переключения между пилотами на лету.

Выполненные и выполняющиеся главные задачи.

  • Провести анализ актуальности проекта.
  • Разработать и утвердить единую художественную концепцию.
  • Построить тестовый уровень.
  • Спроектировать боссфайт.
  • Создать механики для полёта по заданному маршруту с возможностью переключения между пилотами.
  • Спроектировать боссфайт.
  • Создать механики для полёта по заданному маршруту с возможностью переключения между пилотами.
  • Наполнить демо-версию визуальными и звуковыми и т.д. элементами для достижения максимальной презентабельности.
